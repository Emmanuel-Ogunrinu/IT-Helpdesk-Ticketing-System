# KB005: Slow Computer Performance

## Issue
User reports slow login, freezing, or delayed application response.

## Troubleshooting Steps
1. Confirm when issue started.
2. Check CPU, memory, and disk usage.
3. Restart device.
4. Check startup applications.
5. Verify available disk space.
6. Run updates if required.
7. Escalate if hardware failure or malware is suspected.

## Resolution Notes
Unneeded startup applications disabled and device restarted. User confirmed improved performance.
