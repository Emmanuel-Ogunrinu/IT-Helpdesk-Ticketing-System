# KB001: Password Reset

## Issue
User is unable to sign in because they forgot their password or were locked out after multiple failed attempts.

## Troubleshooting Steps
1. Verify the user's identity using approved verification steps.
2. Confirm the affected account and system.
3. Reset the password using the appropriate account management tool.
4. Require password change at next sign-in.
5. Ask the user to test login.
6. Confirm access is restored.

## Resolution Notes
Password reset completed and user confirmed successful login.

## Prevention
Remind user to follow password complexity requirements and avoid reusing previous passwords.
