# KB003: Software Installation Request

## Issue
User requests installation of approved software.

## Troubleshooting / Fulfillment Steps
1. Confirm software name and business need.
2. Verify software is approved.
3. Confirm device compatibility.
4. Install software using approved software center or admin process.
5. Launch application and confirm successful installation with user.
6. Document version installed.

## Resolution Notes
Software installed successfully and user confirmed application opens correctly.
