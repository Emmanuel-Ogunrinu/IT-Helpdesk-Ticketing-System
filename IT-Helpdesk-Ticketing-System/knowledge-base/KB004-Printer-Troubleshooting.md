# KB004: Printer Troubleshooting

## Issue
User cannot print, printer shows offline, or print jobs are stuck.

## Troubleshooting Steps
1. Confirm printer name and location.
2. Check if printer is powered on and connected.
3. Clear stuck print queue if needed.
4. Restart print spooler if applicable.
5. Remove and re-add printer.
6. Print a test page.
7. Escalate to hardware/vendor support if printer hardware failure is suspected.

## Resolution Notes
Print queue cleared and printer re-added. User confirmed successful test print.
