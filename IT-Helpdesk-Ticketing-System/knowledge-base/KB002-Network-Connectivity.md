# KB002: Network Connectivity Issue

## Issue
User cannot connect to Wi-Fi or access internal resources.

## Troubleshooting Steps
1. Confirm whether the issue affects one user or multiple users.
2. Check Wi-Fi connection status.
3. Verify airplane mode is disabled.
4. Restart wireless adapter.
5. Run `ipconfig /release` and `ipconfig /renew` if applicable.
6. Test access to a known website or internal resource.
7. Escalate if multiple users are affected or if network infrastructure may be involved.

## Resolution Notes
Network adapter reset and IP renewed. User confirmed connectivity restored.
